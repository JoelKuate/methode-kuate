# Méthode KUATE

> CLI d'orchestration d'agents IA pour équipes structurées

```bash
npx methode-kuate init
```

## Commandes

| Commande | Description |
|---|---|
| `kuate init` | Initialise la Méthode KUATE dans votre projet |
| `kuate agent list` | Liste les agents disponibles |
| `kuate agent use <nom>` | Copie le prompt d'un agent |
| `kuate workflow list` | Liste les workflows par phase |
| `kuate memory inject` | Génère un bloc contexte pour vos sessions IA |
| `kuate build --target claude` | Exporte pour Claude/ChatGPT/Gemini/Cursor |
| `kuate conseil --agents a,b --topic "..."` | Mode multi-experts |
| `kuate doctor` | Diagnostic de l'installation |

## Phases KUATE

| Phase | Description |
|---|---|
| **K** — Konnaître | Discovery |
| **U** — Unifier | Planning |
| **A** — Architecturer | Design |
| **T** — Transformer | Build |
| **E** — Évaluer | Deliver |
